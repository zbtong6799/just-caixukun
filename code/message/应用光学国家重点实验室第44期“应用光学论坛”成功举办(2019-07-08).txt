应用光学国家重点实验室第44期“应用光学论坛”成功举办(2019-07-08)
　　受应用光学国家重点实验室常务副主任阚瑞峰邀请，中国水稻研究所张健研究员、华中农业大学张建伟研究员、军事医学研究院生物工程研究所赵志虎研究员、安徽省农业科学院魏鹏程研究员等到长春光机所访问交流。7月5日上午9时，第44期应光论坛在学术交流中心第一会议室举办。会议由阚瑞峰主持，阚瑞峰对到访的专家学者表示热烈欢迎，10余位科研人员及研究生参加了本次论坛。
　　与会专家分别作了题为《水稻生殖发育基因克隆与机理解析》、《Building Platinum Standard Reference Genome Sequences for Comparative Genomics》、《基因组三维高级结构与功能》、《水稻高效广适基因组编辑方法》、《核技术应用于植物诱变育种》等报告，分别介绍了水稻生殖发育相关基因的克隆与功能研究，蛋白质翻译后修饰在水稻生殖发育中的调控机理等。同时，详细介绍了作物生物技术应用基础和作物逆境响应调控机理，并针对开展空间水稻的种植研究给出了建议。
　　与会专家针对“封闭生态生命支持系统研发及水稻空间生物学研究”进行了深入的分析，针对空间环境下，水稻全生命周期的监控与培养、生态价值评估、可繁衍性评价等内容进行深入的研讨。通过举办本次论坛，为后续进一步开展合作交流奠定了基础。
